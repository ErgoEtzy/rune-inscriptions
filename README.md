\# Rune Inscriptions



A collection of runic inscription notes in Markdown format.



This repository is meant to be a simple, readable archive of individual inscriptions. Each inscription has its own `.md` file with basic information, text, translation, notes, and sources where available.



\## What this repository contains



\- Individual Markdown files for runic inscriptions

\- Basic inscription information

\- Transliteration

\- Normalized text where available

\- Translation where available

\- Notes on reading, dating, language, or interpretation

\- Source references
