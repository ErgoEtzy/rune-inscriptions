Rune Inscriptions

=================



A collection of runic inscription notes in Markdown format, made for importing in Obsidian.



What this repository contains

\-----------------------------



\- Individual files for runic inscriptions

\- Basic inscription information

\- Transliteration

\- Normalized text where available

\- Translation where available

\- Notes on reading, dating, language, or interpretation

\- Source references



Repository structure

\--------------------



```text

rune-inscriptions/

&#x20; README.md

&#x20; index.md

&#x20; inscriptions/

&#x20;   individual inscription files

```



File format

\-----------



Each inscription file can follow this general structure:



```text

Inscription ID — Name



Basic information



ID:

Name:

Location:

Date/period:

Rune row/script:

Object type:



Transliteration



Normalized text



Translation



Notes



Sources

```



Notes on uncertainty

\--------------------



Runic inscriptions are often difficult to read and interpret. Readings, datings, normalizations, and translations may vary between scholars.
